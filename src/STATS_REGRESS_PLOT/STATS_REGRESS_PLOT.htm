Plots Useful for Regression Relationships

This command displays a set of plots that may be useful in assessing regression type relationships between one or more dependent variables and a set of explanatory variables, although they are not limited to this use. These charts are not partial plots.

Vertical-Axis Variables Select one or more scale-level dependent variables. Each will be plotted against each of the horizontal-axis variables. Besides dependent variables, it may be useful to save regression residuals and use those variables here in order to assess the adequacy of the regression.

Horizontal-Axis Variables Select one or more variables to be plotted against. If a variable has a scale measurement level, the graph will be a scatterplot. If it is categorical, the graph will be a bar chart, a line chart, or a boxplot.

The following controls apply only to points.

Color by Choose a grouping variable to color scatterplot points by. It can be scale or categorical.

Size by Choose a categorical or scale variable to use to determine the size of points in a scatterplot. It may be useful to choose a leverage or influence variable from a regression here.

Shape by Choose a categorical variable to determine the shape of points in a scatterplot.

Label by Choose a variable to label points in a scatterplot or boxplot.

Options

Chart Title Enter a title for the charts. The title will be displayed only for unpaneled charts without bordering boxplots. It will also appear in the Settings table accompanying the charts.

Scatterplot Fit Lines Fit lines can be one or more of linear, quadratic, cubic, or loess. Each type uses a different line pattern.

Grouping Fit lines can be for all points together or, if there is a categorical color variable, there can be separate lines for each group.

Categorical Variable Plots Categorical horizontal plots can be bar charts of means, line charts of means, or boxplots.

Bin points in scatterplots Check this box to group points into bins. This can be particularly helpful if there are many points or if you want to focus on isolated points.

Bordered boxplots for scatterplots If charts are not paneled, check this box to include boxplots for the vertical and horizontal variables in each scatterplot.

Chart Layout Parameters By default, each chart is separate. You can specify the number of charts to panel across. If charts are paneled, legends and bordering boxplots are not displayed, but legend information appears in the settings table.

Page Size Enter vertical and horizontal page size values to change the size of a chart. A row of paneled charts counts as one chart when sizing. Both vertical and horizontal size must be specified if either is given. Measures are in inches.

Parameters for Multiple Charts Per Row If chart labels or legends are large, they may be clipped or spill over into another chart with the default settings. These parameters can be used to improve this behavior. Values are in percent.

Additional Features

This dialog generates syntax for the STATS REGRESS PLOT extension command. To display help for this command, run the following syntax:

STATS REGRESS PLOT /HELP.

Requirements

This command requires the Python Essentials and the STATS REGRESS PLOT extension command. You can download these from the SPSS Community at www.ibm.com/developerworks/spssdevcentral.
