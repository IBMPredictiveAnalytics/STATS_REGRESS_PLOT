Gráficos útiles para las relaciones de regresión

Este comando muestra un conjunto de gráficos que pueden resultar útiles para evaluar las relaciones de los tipos de regresiones entre una o más variables dependientes y un conjunto de variables explicatorias, aunque no se limitan a este uso. Estos gráficos no son gráficos parciales.

Variables del eje vertical Seleccione una o más variables dependientes a nivel de escala. Cada una se trazará contra una de las variables del eje horizontal. Aparte de las variables dependientes, puede resultar útil guardar los residuos de regresión y utilizar esas variables aquí para evaluar la idoneidad de la regresión.

Variables del eje horizontal Seleccione una o más variables contra las que se trazarán las otras. Si una variable tiene un nivel de medición de escala de escala, el gráfico será un diagrama de dispersión. Si es categórico, el gráfico será un gráfico de barras o un diagrama de caja.

Los controles siguientes se aplican únicamente a los puntos.

Colorear por Elija una variable de agrupación por la que colorear los puntos del diagrama de dispersión. Puede ser de escala o categórica.

Tamaño por Elija una variable categórica o de escala que utilizar para determinar el tamaño de los puntos de un diagrama de dispersión. Puede resultar útil elegir una variable de optimización o de influencia de una regresión de aquí.

Forma por Elija una variable categórica para determinar la forma de los puntos de un diagrama de dispersión.

Etiqueta por Elija una variable para etiquetar los puntos de un diagrama de dispersión o un diagrama de caja.

Opciones

Título del gráfico Especifique un título para los gráficos. El título sólo se mostrará en los gráficos no panelados sin diagramas de dispersión adyacentes. También aparecerá en la tabla de configuración que acompaña a los gráficos.

Líneas de ajuste de diagrama de dispersión Las líneas de ajuste pueden ser lineales, cuadráticas, cúbicas o loess. Cada tipo utiliza un patrón de línea distinto.

Agrupación Las líneas de ajuste pueden ser para todos los puntos o, si hay una variable de color categórica, puede haber líneas separadas para cada grupo.

Gráficos de variables categóricas Los gráficos horizontales categóricos pueden ser diagramas de barras de medias, diagramas de líneas de medias o diagramas de caja.

Agrupar puntos en los gráficos de dispersión Marque esta casilla para agrupar los puntos en clases. Esto puede resultar especialmente útil si hay muchos puntos o si desea concentrar la atención en puntos aislados.

Gráficos de caja adyacentes a los diagramas de dispersión Si los gráficos no están panelados, puede marcar este recuadro para incluir diagramas de caja para las variables verticales y horizontales de cada diagrama de dispersión.

Parámetros de diseño de gráficos De forma predeterminada, cada gráfico va aparte. Puede especificar el número de gráficos que desea panelar. Si los gráficos están panelados, las leyendas y los diagramas de caja adyacentes no se muestran, sino que la información de leyenda aparece en la tabla de configuración.

Tamaño de página Especifique los valores de tamaño horizontal y vertical de página para cambiar el tamaño de un gráfico. Una fila de gráficos panelados cuenta como un gráfico al definir el tamaño. Se debe especificar tanto el tamaño vertical como el horizontal si se especifica uno de los dos. Las medidas son en pulgadas.

Parámetros para varios gráficos por fila Si las etiquetas o loas leyendas de los gráficos son largas, con la configuración predeterminada puede ocurrir que queden cortadas o que ocupen parte de otro gráfico.
