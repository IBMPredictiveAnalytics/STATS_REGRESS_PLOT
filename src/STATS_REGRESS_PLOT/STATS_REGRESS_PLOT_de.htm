Nützliche Diagramme für Regressionsbeziehungen

Dieser Befehl zeigt eine Reihe von Diagrammen an, die für die Auswertung von Regressionstypbeziehungen zwischen einer oder mehreren abhängigen Variablen und einem Satz erklärender Variablen nützlich sein können. Sie sind jedoch nicht auf diese Verwendung beschränkt. Diese Diagramme sind keine Teildiagramme.

Vertikale Achsenvariablen Wählen Sie mindestens eine abhängige Variable mit metrischem Niveau aus. Jede Variable wird gegen die horizontalen Achsenvariablen abgebildet. Neben abhängigen Variablen kann es nützlich sein, Regressionsresiduen zu speichern und diese Variablen hier zu verwenden, um die Angemessenheit der Regression zu bewerten.

Horizontale Achsenvariablen Wählen Sie mindestens eine Variable für die Gegenachse aus. Wenn eine Variable ein metrisches Messniveau hat, handelt es sich bei der Kurve um ein Streudiagramm. Wenn es sich um eine kategoriale Variable handelt, ist die Kurve ein Balkendiagramm, Liniendiagramm oder Boxplot.

Folgende Steuerelemente gelten nur für Punkte.

Farbe nach Wählen Sie eine Gruppierungsvariable aus, nach der die Farbe für die Punkte in einem Streudiagramm bestimmt werden soll. Diese kann metrisch oder kategorial sein.

Größe nach Wählen Sie eine kategoriale oder metrische Variable aus, nach der die Größe der Punkte in einem Streudiagramm bestimmt werden soll. Es kann nützlich sein, hier eine Einflussvariable aus einer Regression zu wählen.

Form nach Wählen Sie eine kategoriale Variable aus, nach der die Form der Punkte in einem Streudiagramm bestimmt werden soll.

Beschriften anhand Wählen Sie eine Variable aus, nach der die Punkte in einem Streudiagramm oder Boxplot beschriftet werden sollen.

Optionen

Titel für Diagramm Geben Sie einen Titel für die Diagramme ein. Der Titel wird nur für Diagramme ohne Feldaufteilung und ohne angrenzende Boxplots angezeigt. Er wird zudem in der Einstellungstabelle zu den Diagrammen angegeben.

Anpassungslinien für Streudiagramm Anpassungslinien können eine oder mehrere Linien vom Typ linear, quadratisch, kubisch oder Loess sein. Jeder Typ hat ein anderes Linienmuster.

Gruppieren Anpassungslinien können für alle Punkte zusammen verwendet werden, oder, wenn eine kategoriale Farbvariable ausgewählt ist, können für jede Gruppe separate Linien verwendet werden.

Kategoriale Variablendiagramme Bei kategorialen horizontalen Diagrammen kann es sich um Balkendiagramme der Mittelwerte, Liniendiagramme der Mittelwerte oder Boxplots handeln.

Klassenpunkte in Streudiagrammen Wählen Sie dieses Kontrollkästchen aus, um Punkte in Klassen zu gruppieren. Dies kann besonders nützlich sein, wenn viele Punkte vorhanden sind oder wenn Sie sich auf einzelne Punkte konzentrieren möchten.

Angrenzende Boxplots für Streudiagramme Wenn die Diagramme nicht in Felder aufgeteilt sind, wählen Sie dieses Kontrollkästchen aus, um für die vertikalen und horizontalen Variablen in jedem Streudiagramm Boxplots anzuzeigen.

Diagrammlayoutparameter Standardmäßig wird jedes Diagramm separat dargestellt. Sie können angeben, wie viele Diagramme in einem Feld dargestellt werden sollen. Wenn Diagramme Felder enthalten, werden keine Legenden oder angrenzenden Boxplots angezeigt, die Legendeninformationen werden jedoch in der Einstellungstabelle aufgeführt.

Seitenlänge Geben Sie einen Wert für die vertikale und horizontale Seitenlänge ein, um die Größe eines Diagramms zu ändern. Eine Reihe von Diagrammen in einem Feld zählt bei der Größenbestimmung als ein Diagramm. Die vertikale und die horizontale Länge müssen immer zusammen angegeben werden. Als Maßeinheit wird Zoll verwendet.

Parameter für mehrere Diagramme pro Zeile Wenn Diagrammbeschriftungen oder -legenden lang sind, werden sie bei Verwendung der Standardeinstellungen möglicherweise abgeschnitten oder erstrecken sich in ein anderes Diagramm hinein. Mithilfe dieser Parameter kann dieses Verhalten verbessert werden. Die Werte werden in Prozent angegeben.

Weitere Funktionen

In diesem Dialogfeld wird Syntax für den Erweiterungsbefehl STATS REGRESS PLOT generiert. Führen Sie folgende Syntax aus, um Hilfe für diesen Befehl anzuzeigen:

STATS REGRESS PLOT /HELP.

Anforderungen

Für diesen Befehl sind Python Essentials und der Erweiterungsbefehl STATS REGRESS PLOT erforderlich. Sie können diese von der SPSS Community unter www.ibm.com/developerworks/spssdevcentral herunterladen.
