Tracés utiles pour les relations de régression

Cette commande affiche un ensemble de tracés qui peuvent être utiles pour évaluer les relations de type régression entre une ou plusieurs variables dépendantes et un ensemble de variables explicatives bien que ces tracés ne soient pas limités à cette utilisation. Ces graphiques ne sont pas partiels.

Variables d'axe vertical Sélectionnez une ou plusieurs variables dépendantes du niveau d'échelle. Chacune sera tracée en fonction de chaque variable d'axe horizontal. En plus des variables dépendantes, il peut être utile d'enregistrer les résiduels de régression et d'utiliser ces variables ici pour évaluer l'adéquation de la régression.

Variables d'axe horizontal Sélectionnez une ou plusieurs variables en fonction desquelles effectuer un tracé. Si une variable a un niveau de mesure d'échelle, le graphique sera un nuage de points. Si la variable est catégorielle, le graphique sera un graphique à barres, linéaire ou diagramme à surfaces.

Les contrôles suivants s'appliquent uniquement aux points.

Couleur par Choisissez une variable de regroupement avec laquelle colorer les points du nuage de points. Il peut s'agir d'une variable d'échelle ou catégorielle.

Taille par Choisissez une variable d'échelle ou catégorielle à utiliser pour déterminer la taille des points d'un nuage de points. Il peut être utile de choisir ici un point de levier ou une variable influente depuis une régression.

Forme par Choisissez une variable catégorielle pour déterminer la forme des points d'un nuage de points.

Libellé par Choisissez une variable pour étiqueter les points d'un nuage de points ou d'un diagramme à surfaces.

Options

Titre du graphique Saisissez un titre pour les graphiques. Le titre s'affichera uniquement pour les graphiques qui ne sont pas en panneaux et dépourvus de diagrammes à surfaces avec bordures. Il apparaîtra également dans la table Paramètres qui accompagne les graphiques.

Courbes d'ajustement de nuages de points Les courbes d'ajustement peuvent être une ou plusieurs courbes linéaires, quadratiques, cubiques ou de loess. Chaque type utilise un motif de courbe différent.

Regroupement Les courbes d'ajustement peuvent être pour tous les points ou, s'il y a une variable de couleur catégorielle, elles peuvent être des courbes séparées pour chaque groupe.

Graphiques de variables catégorielles Les graphiques horizontaux catégoriels peuvent être des graphiques à barres de moyennes, des graphiques linéaires de moyennes ou des diagramme à surfaces.

Regrouper les points en classes dans les nuages de points Cochez cette case pour regrouper les points en classes. Cela peut être particulièrement utile s'il y a de nombreux points ou si vous souhaitez vous focaliser sur des points isolés.

Diagramme à surfaces avec bordures pour les nuages de points Si les graphiques ne sont pas sous forme de panels, cochez cette case pour inclure des diagramme à surfaces pour les variables verticales et horizontales dans chaque nuage de points.

Paramètres de la présentation des graphiques Par défaut, chaque graphique est représenté séparément. Vous pouvez spécifier le nombre de graphiques à présenter sous forme de panel. Si les graphiques sont sous forme de panels, les légendes et les diagrammes à surfaces avec bordures n'apparaissent pas, mais des informations sur les légendes s'affichent dans la table des paramètres.

Taille de page Saisissez les valeurs de la taille de la page verticale et horizontale pour modifier la taille d'un graphique. Une ligne de graphiques en panneaux compte comme un seul graphique lors de l'ajustement de la taille. Les tailles verticale et horizontale doivent être spécifiées si elles sont indiquées. Les mesures sont en pouce.

Paramètres de plusieurs graphiques par ligne Si des libellés ou des légendes de graphiques sont de taille importante, ils peuvent être rognés ou transmis à un autre graphique avec les paramètres par défaut. Ces paramètres peuvent être utilisés pour améliorer ce comportement. Les valeurs sont en pourcentage.

Fonctions supplémentaires

Cette boîte de dialogue génère une syntaxe pour la commande d'extension STATS REGRESS PLOT. Pour afficher l'aide de cette commande, exécutez la syntaxe suivante :

STATS REGRESS PLOT /HELP.

Conditions requises

Cette commande nécessite Python Essentials et la commande d'extension STATS REGRESS PLOT. Vous pouvez les télécharger depuis la communauté SPSS à l'adresse www.ibm.com/developerworks/spssdevcentral.
