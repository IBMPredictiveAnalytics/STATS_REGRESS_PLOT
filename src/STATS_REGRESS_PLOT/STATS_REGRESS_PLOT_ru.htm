Диаграммы, полезные при оценке взаимоотношений регрессии

Эта команда выводит набор диаграмм, которые могут быть полезны при оценке взаимоотношений типов регрессии между одной или несколькими зависимыми переменными и набором объясняющих переменных, хотя этим их использование не ограничивается. Эти диаграммы не являются частными графиками.

Переменные вертикальной оси Выберите одну или несколько зависимых количественных переменных. Каждая из них будет показана в зависимости от всех переменных горизонтальной оси. Кроме зависимых переменных, может иметь смысл сохранить регрессионные остатки и использовать здесь эти переменные для оценки адекватности регрессии.

Переменные горизонтальной оси Выберите одну или несколько переменных для графического сопоставления. Если у переменной есть уровень шкалы измерений, то график будет диаграммой рассеяния. Для категориальной переменной график будет столбчатой диаграммой, линейной диаграммой или ящичной диаграммой.

Указанные ниже элементы управления применяются только к точкам.

Цвет по Выберите группирующую переменную для задания цвета точек диаграммы рассеяния. Она может быть количественной или категориальной.

Размер по Выберите категориальную или количественную переменную, чтобы задать размер точек на диаграмме рассеяния. Здесь может оказаться полезным воспользоваться переменной влияния из регрессии.

Форма по Выберите категориальную переменную, чтобы задать форму точек на диаграмме рассеяния.

Метка по Выберите переменную, чтобы пометить точки на диаграмме рассеяния или ящичной диаграмме.

Параметры

Заголовок диаграммы Введите заголовок для диаграмм. Этот заголовок будет выводиться только для диаграмм без панелей и окаймляющих ящичных диаграмм. Он будет выводиться также в таблице Параметры, прилагаемой к диаграммам.

Аппроксимирующие линии диаграммы рассеяния Аппроксимирующих линий может быть одна или несколько - с линейной, квадратичной, кубической зависимостью или использующих локально взвешенный МНК. Для каждого типа используется свой узор линии.

Группировка Аппроксимирующие линии могут строиться для всего набора точек или, при наличии категориальной цветовой переменной, для каждой группы может строиться своя линия.

Графики категориальных переменных Категориальные горизонтальные графики могут представлять собой столбчатые диаграммы средних, линейные диаграммы средних или ящичные диаграммы.

Точки интервалов на диаграммах рассеяния Включите этот переключатель, чтобы группировать точки по интервалам. Это может быть особенно полезно при большом количестве точек или если нужно сосредоточиться на отдельных точках.

Ящичные диаграммы для диаграмм рассеяния Если диаграммы не содержат панелей, включите этот переключатель, чтобы добавить в каждую диаграмму рассеяния ящичные диаграммы для вертикальной и горизонтальной переменной.

Параметры размещения диаграмм По умолчанию каждая диаграмма выводится отдельно. Вы можете задать число диаграмм, выводимых на панели. Если диаграммы размещены на панели, то легенды и окаймляющие ящичные диаграммы не показываются, а информация легенд выводится в таблице параметров.

Размер страницы Введите размер страницы по вертикали и горизонтали, чтобы изменить размер диаграммы. При изменении размера ряд диаграмм с панелями считается одной диаграммой. Если дан только один из размеров, по вертикали или по горизонтали, необходимо указать оба. Измерения даются в дюймах.

Параметры для ряда из нескольких диаграмм Если у диаграммы большие метки или легенда, они могут быть обрезаны или зайти на другую диаграмму с параметрами по умолчанию. Эти параметры можно использовать для улучшения ситуации. Значения указываются в процентах.

Дополнительные возможности

Это диалоговое окно генерирует синтаксис для команды расширения STATS REGRESS PLOT. Чтобы показать справку для этой команды, выполните следующий синтаксис:

STATS REGRESS PLOT /HELP.

Технические требования

Для выполнения этой команды требуется Python Essentials и команда расширения STATS REGRESS PLOT. Можно загрузить их из SPSS Community по адресу www.ibm.com/developerworks/spssdevcentral.
