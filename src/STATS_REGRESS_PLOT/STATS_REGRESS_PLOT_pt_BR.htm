Gráficos Úteis para Relacionamentos de Regressão

Esse comando exibe um conjunto de gráficos que podem ser úteis para avaliar os relacionamentos do tipo de regressão entre uma ou mais variáveis dependentes e um conjunto de variáveis explicativas, apesar de não estarem limitadas a esse uso. Esses gráficos não são gráficos parciais.

Variáveis de Eixo Vertical Selecione uma ou mais variáveis dependentes em nível de escala. Cada uma será plotada com relação a cada uma das variáveis do eixo horizontal. Além das variáveis dependentes, pode ser útil salvar os resíduos da regressão e usar essas variáveis aqui para avaliar a adequação da regressão.

Variáveis de Eixo Horizontal Selecione uma ou mais variáveis com relação às quais criar gráfico. Se uma variável tiver um nível de medição de escala, o gráfico será de dispersão. Se for categórico, o gráfico será de barras, de linhas ou boxplot.

Os seguintes controles aplicam-se apenas a pontos.

Colorir por Escolha uma variável de agrupamento de acordo com a qual colorirá os pontos do gráfico de dispersão. Pode ser de escala ou categórica.

Dimensionar por Escolha uma variável de escala ou categórica para usar para determinar o tamanho dos pontos em um gráfico de dispersão. Pode ser útil escolher uma variável de influência ou alavanca de uma regressão aqui.

Formatar por Escolha uma variável categórica para determinar o formato dos pontos em um gráfico de dispersão.

Rotular por Escolha uma variável para rotular pontos em um gráfico de dispersão ou boxplot.

Opções

Título do Gráfico Insira um título para os gráficos. O título será exibido apenas para gráficos sem painéis e sem boxplots ao redor. Também aparece na tabela Configurações que acompanha os gráficos.

Linhas de Ajuste do Gráfico de Dispersão As linhas de ajuste podem ser uma ou mais de linear, quadrática, cúbica ou loesse. Cada tipo usa um padrão de linha diferente.

Agrupamento Linhas de ajuste podem ser para todos os pontos juntos ou, se houver uma variável de cor categórica, poderá haver linhas separadas para cada grupo.

Gráficos de Variável Categórica Gráficos horizontais categóricos podem ser gráficos de barras de médias, gráficos de linha de média ou boxplots.

Categorizar Pontos em Gráficos de Dispersão Marque esta caixa para agrupar pontos em categorias. Isso pode ser particularmente útil se houver muitos pontos ou se você desejar focar em pontos isolados.

Boxplots com Bordas para Gráficos de Dispersão Se os gráficos não tiverem painéis, marque esta caixa para incluir boxplots para as variáveis verticais e horizontais em cada gráfico de dispersão.

Parâmetros de Layout do Gráfico Por padrão, cada gráfico é separado. Você pode especificar o número de gráficos através dos quais fazer um painel. Se os gráficos estiverem em painel, as legendas e boxplots ao redor não são exibidos, mas as informações de legenda aparecem na tabela de configurações.

Tamanho da Página Insira valores de tamanho de página verticais e horizontais para alterar o tamanho de um gráfico. Uma linha de gráficos em painel conta como um gráfico no dimensionamento. Os tamanhos vertical e horizontal devem ser especificados se qualquer um deles for fornecido. As medidas são em polegadas.

Parâmetros para vários Gráficos por Linha Se as legendas ou os rótulos do gráfico forem grandes, eles poderão ser recortados ou se estender até outro gráfico com as configurações padrão. Esses parâmetros podem ser usados para melhorar esse comportamento. Os valores são em percentagem.

Recursos adicionais

Esta caixa de diálogo gera sintaxe para o comando de extensão STATS REGRESS PLOT. Para exibir ajuda para este comando, execute a seguinte sintaxe:

STATS REGRESS PLOT /HELP.

Requisitos

Esse comando requer o Python Essentials e o comando de extensão STATS REGRESS PLOT. Você pode fazer o download deles na SPSS Community em www.ibm.com/developerworks/spssdevcentral.
